# 居酒屋バイト シフト管理ツール

自分の居酒屋のアルバイトスタッフを対象とした、簡易的なシフト管理CLIツールです。

## 概要

スタッフ情報の登録・管理と、日ごとのシフト入力・表示ができます。  
C言語での論理構築の練習として作成したものであり、UIや見栄えを重視した設計にはなっていません。

## 機能

- スタッフの追加・一覧表示・削除・情報変更
- シフトの追加・表示・変更・削除
- 重複シフトのチェック
- 終電時刻によるシフト登録制限
- 日付またぎシフトの対応（例：23:00〜01:00）

## ビルド・実行方法

```bash
gcc parttime_shift.c -o parttime_shift
./parttime_shift
```

Windowsの場合：

```bash
gcc parttime_shift.c -o parttime_shift.exe
parttime_shift.exe
```

## 動作環境

- C99以降に対応したCコンパイラ（GCC推奨）
- Windows / Linux / macOS

## 注意事項

- データはプログラム終了時にすべて消えます（ファイル保存機能なし）
- UIや操作性は最低限です
- 閏年には対応していません（2月は常に28日扱い）
